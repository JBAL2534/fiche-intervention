Dossier de destination : choisir / changer
Commandes vocales
  • Identité : correspondant …, nom …, prénom …, dossier …, date …
  • Intervention : description … (ou touchez le cadre et dictez avec le clavier)
  • Avulsion : avulsion oui / avulsion non
  • Implants : implant active 16, implant parallel 21 23 25 (plusieurs dents = plusieurs implants). Ø/longueur à préciser à la main.
  • Matériel : dites le nom → pilier provisoire, pins, creos zéro cinq, purgo zéro cinq, gto, biobank, txt 200, contre-angle rouge, pièce à main, guide labo, empreinte alv/wyart, forets versah
  • PRF (nombre variable) : six tubes rouges, deux tubes blancs
  • Autre élément : ajoute … (ex. « ajoute vis de couverture »)
  • Correction : décoche …, efface nom
